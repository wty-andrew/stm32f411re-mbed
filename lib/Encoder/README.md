# Connection

| Wire Color | Function                           |
|------------|------------------------------------|
| Green      | Hall Sensor GND                    |
| Blue       | Hall Sensor Vcc (5V)               |
| Yellow     | Hall Sensor A Vout (Interrupt Pin) |
| White      | Hall Sensor B Vout (Digital Pin)   |
